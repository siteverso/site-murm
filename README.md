# Murmurinho

Protótipo funcional de rede social feito em Astro.

## Executar

```bash
npm install
npm run dev
```

Abra `http://localhost:4324`.

## Estado atual

Os dados de login, perfil, murmúrios, votos, respostas e compartilhamentos são salvos no `localStorage` do navegador. Isso permite validar toda a experiência antes da criação da API e do banco.
